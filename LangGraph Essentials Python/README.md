# LangGraph Essentials - Python

This folder contains tasks for LangGraph Essentials - Python course offered by [LangChain Academy](https://academy.langchain.com/)

The course gives an introduction to building applications with LangGraph covering the following topics:

- [Nodes](L1_nodes.ipynb)
- [Edges](L2_edges.ipynb)
- [Conditional Edges & Memory](L3-L4_cedges_memory.ipynb)
- [Interrupts](L5_interrupt.ipynb)
- [Email Agent](L6_EmailAgent.ipynb)

> **Disclaimer**  
> This repository adapts the [lca-langgraph-essentials](https://github.com/langchain-ai/lca-langgraph-essentials) course materials to be used with Amazon Bedrock LLMs. It also has other minor adaptations and extra annotations.
