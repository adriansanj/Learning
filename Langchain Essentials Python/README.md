# LangChain Essentials - Python

This folder contains tasks for LangChain Essentials - Python course offered by [LangChain Academy](https://academy.langchain.com/)

The course gives an introduction to the creation of agents using LangChain, LangGraph and LangSmith covering the following topics:

- [Easy agent creation](L1_fast_agent.ipynb)
- [Messages](L2_messages.ipynb)
- [Streaming](L3_streaming.ipynb)
- [Tools](L4_tools.ipynb)
- [Tools with MCP](L5_tools_with_mcp.ipynb)
- [Memory](L6_memory.ipynb)
- [Structured Output](L7_structuredOutput.ipynb)
- [Dynamic Prompts](L8_dynamic.ipynb)
- [Human In The Loop](L9_HITL.ipynb)

> This repository adapts the [lca-langchainV1-essentials](https://github.com/langchain-ai/lca-langchainV1-essentials) repository to be used with Amazon Bedrock LLMs. It also has other minor adaptations and extra annotations.
